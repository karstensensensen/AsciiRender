Detailed Description

event class for key pressed events

stores the keycode that generated the event, as well as the repeat flag for the key event. the repeat flag is set to true if the keycode was generated because it was held down for an extended period of time.

Categories: Asciir::CategoryInput, Asciir::CategoryKeyboard
Type: EventType::KeyPressed
