Detailed Description

template<typename T>
struct Asciir::arTensor3D< T >

A wrapper for arMatrix that overloads acsess functions to accept 3 indexes instead of 2.

implementation details:
resize and operator() gets the element by using w*h as the first index and d as the second index. everything else functions as usual
