Detailed Description

small class storing the path to an audio file, as well as how it should be loaded.

if toggleFX is set to true, the audio file will be loaded into memory, when an AudioPlayer loads it. Otherwise, it will be read directly from disk.

the audio file can be one of these formats: WAV OGG FLAC ?? MP3 ??

the AudioSource is not required to exists whilst the AudioPlayer has the sound loaded. It is only required on the load function. this also means any subsequent changes to the AudioSource will not be reflected in the AudioPlayer, unless it is reloaded.
