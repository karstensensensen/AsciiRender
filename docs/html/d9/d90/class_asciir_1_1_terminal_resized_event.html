Detailed Description

event class for terminal resize events.

a maximum of 1 resize event is generated per update.
This event will act as if the previous size before the update has been resized directly to the new size during the update, ignoring any other sizing modifications inbetween.

Categories: Asciir::CategoryTerminal
Type: EventType::TerminalResized
