Detailed Description

wrapper class of FileTexture specialized for reading sprite sheets.

use the constructor or the provided methods to configure the sprite sheet layout. one configured use setSprite() to set the sprite tile that readTile() should use.

when passing to Renderer::submit(), the getSprite() method should be used, as it only copies the selected sprite,instead of the entire sprite sheet, into the render buffer.

when a sprite sheet is tiled, only the active sprite is tiled.
